# 2 Tắt DHCP trên pfSense

Trước tiên phải tránh việc có hai DHCP Server cùng cấp phát IP.

Vào:
``
Services
    ↓
DHCP Server
```

Bỏ chọn:

```
Enable DHCP Server
``
Nhấn:

```
Save
Apply Changes
```
Sau bước này, pfSense không còn cấp IP.

# 6 Exclusion Range

Add Exclusion and Delay (Đây là các IP không được DHCP cấp).

Ví dụ:
```
Thiết bị	IP
pfSense	192.168.90.254
DC01	192.168.90.10
Switch Cisco	192.168.90.2
Printer	192.168.90.20-29
Camera	192.168.90.30-39
```
Có thể tạo Exclusion:

```
192.168.90.20

↓

192.168.90.39
```

Nếu chưa có thiết bị thì có thể để trống và cấu hình sau.

# 14. Kiểm tra từ Client

Đặt card mạng ở chế độ:

Obtain an IP address automatically

Sau đó mở CMD:

```
ipconfig /release
ipconfig /renew
```

Kiểm tra:

```
ipconfig /all
```
Kết quả mong muốn:
```
IPv4 Address . . . . . : 192.168.90.50-200
Subnet Mask . . . . . : 255.255.255.0
Default Gateway . . . : 192.168.90.254
DHCP Server . . . . . : 192.168.90.10
DNS Servers . . . . . : 192.168.90.10
Connection-specific DNS Suffix :
seta-international.vn
```

# 15. Reservation (IP cố định theo MAC)

Nếu muốn một thiết bị luôn nhận cùng một IP (ví dụ máy in hoặc PC của kế toán):

Trong DHCP Manager:
```
IPv4
   └── Scope [192.168.90.50 - 192.168.90.200]
        └── Reservations
```
Chuột phải Reservations → New Reservation...

Nhập:

```
Reservation name: Printer-01
IP address: 192.168.90.21
MAC address: 00-11-22-33-44-55
Supported types: Both
Nhấn Add.
```
Thiết bị đó sẽ luôn nhận đúng IP này khi xin DHCP.

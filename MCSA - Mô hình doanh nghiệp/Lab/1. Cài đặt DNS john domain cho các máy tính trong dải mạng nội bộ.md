Tài liệu triển khai Active Directory Domain Services (AD DS) và DNS trên Windows Server 2019
----

# 1. Mục tiêu

Triển khai Windows Server 2019 làm Domain Controller cho hệ thống.

Domain nội bộ:
```
seta-international.vn
```
Máy chủ đảm nhiệm các dịch vụ:

- Active Directory Domain Services (AD DS)
- DNS Server
- DHCP Server (tùy chọn)
- Quản lý người dùng và máy tính trong Domain

# 2. Mô hình triển khai
```
                    INTERNET
                        │
                 pfSense Firewall
               192.168.90.254/24
                        │
        ---------------------------------
        │                               │
 Windows Server 2019               Client Windows
 Domain Controller                 Windows 10/11
 192.168.90.10                     DHCP hoặc IP tĩnh

```

# 3. Thông tin cấu hình

Thiết bị	IP
- Domain Controller	192.168.90.10
- Gateway	192.168.90.254
- DNS	192.168.90.10
- Domain	seta-international.vn
- NetBIOS	SETA-INTERNATIONAL

## Bước 1. Đặt IP tĩnh cho Windows Server

Mở
```
Control Panel

↓

Network and Sharing Center

↓

Change adapter settings

↓

Ethernet

↓

Properties

↓

Internet Protocol Version 4 (TCP/IPv4)
```

Chọn
```
Use the following IP address
```

Nhập

| Thông số        | Giá trị        |
| --------------- | -------------- |
| IP Address      | 192.168.90.10  |
| Subnet Mask     | 255.255.255.0  |
| Default Gateway | 192.168.90.254 |
| Preferred DNS   | 192.168.90.10  |

Không sử dụng
```
8.8.8.8
```
làm DNS của Domain Controller.

## Bước 3. Kiểm tra cấu hình mạng

Mở CMD
```
ipconfig /all
```

Kiểm tra
```
IP chính xác
DNS là 192.168.90.10
Gateway là 192.168.90.254
```

## Bước 4. Cài đặt Active Directory Domain Services

Mở
```
Server Manager

↓

Add Roles and Features

↓

Chọn: Role-based or feature-based installation

↓

Chọn máy chủ

↓

Đánh dấu

Active Directory Domain Services

Windows sẽ yêu cầu cài thêm

DNS Server

↓

Chọn

Add Features

↓

Next

↓

Install
```

## Bước 5. Promote thành Domain Controller

Sau khi cài đặt xong.

Trong Server Manager sẽ xuất hiện

```
Promote this server to a domain controller
```
Nhấn vào.

## Bước 11. Kiểm tra Domain

Đăng nhập
```
SETA-INTERNATIONAL\Administrator
```

```
administrator@seta-international.vn
```

```
echo %userdomain%
```

```
SETA-INTERNATIONAL
```

Mở
```

↓

```

```

```

```

_tcp

_udp

DomainDnsZones

ForestDnsZones
```

## Bước 13. Tạo Reverse Lookup Zone

```
DNS

↓

Reverse Lookup Zones

↓

New Zone
```

Chọn

```
Primary Zone

↓

IPv4

↓

Network ID

192.168.90

↓

Finish
```

## Bước 14. Cấu hình DNS Forwarder

Mở
```
DNS

↓

↓

Properties

↓

Forwarders
```

Thêm

```
8.8.8.8

1.1.1.1
```

↓

Nhấn OK

## Bước 15. Kiểm tra phân giải DNS

CMD

```
nslookup google.com
```
Nếu trả về địa chỉ IP thì DNS hoạt động đúng.

## Bước 16. Join máy Client vào Domain

Trên máy Windows 10/11:

Đặt DNS
```
192.168.90.10
```
hoặc nhận DHCP từ Domain.

Mở

```
System

↓

Rename this PC (Advanced)

↓

Change
```

Chọn

```
Domain
```

Nhập

```
seta-international.vn
```
↓

Nhập tài khoản
```
Administrator

hoặc

administrator@seta-international.vn
```

↓

Nhập Password

↓

OK

Nếu xuất hiện thông báo:
```
Welcome to the seta-international.vn domain.
```
thì máy đã gia nhập Domain thành công. Khởi động lại máy để hoàn tất.
